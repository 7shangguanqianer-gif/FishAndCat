# C3 · 能耗与安全叙事

## 一、年化能耗经济账

L2混合流的sim净机械功情景换算下，AWRA-LS布局较题面基线(seq)年化能耗从 **545.4 kWh** 降至 **278.9 kWh**，降幅48.9%，年省电 **266.5 kWh**，折合电费省213元；按2023全国平均因子0.5306kgCO₂/kWh（生态环境部公告2025年第47号）减排约 **141 kg/年**。模型只计提升mgh+水平μmgd/η，未计待机、辅助、驱动器和动态损耗。

**限定说明（呼应D-4）**：上述kWh是sim净机械功情景换算，**并非AC500/电表实测功耗**；绝对量不作工程承诺，相对比较才是核心。

## 二、Worker-Safety 语义：重货低层 = 人工介入更安全

安全溢价账（`sim\out\oracle_gap_summary.txt:16-19`）：holistic 检测档（主打 AWRA、题面 C6 重货低层导向）相较 speed 档，重货平均层高 **0.57 vs 3.71**（低 6.6 倍）、能耗代理 **17685 vs 25891**（低 32%）。

叙事：sim中的holistic规则主动牺牲约24.7%纯速度gap，换重货集中低层堆放——低层意味着人工/AGV介入时提升高度更低、坠落风险窗口更短；同时与sim能耗代理下降同向。speed/lex/holistic三档数字仍只是sim对照；当前ST源码已有holistic/lex切档与CB/TOB执行器，但尚未AB复测，不能写成已验证的在线三档效果。

**边界声明**：本段仅用能耗代理量与heavy_tier两个sim内部指标。lex档98.5%只在`excess_fail=0`可比域成立；即使当前ST源码已有lex/CB/TOB，也因未AB复测而不构成PLC证据。

# C5 · δ（位置价值预留项）管理语义包装

**① 技术定义与复核结果**：δ是评分函数中的位置价值预留权重，默认0.15；δ=0只在已占用集合意义上退化为就近。0712加入drain phase后，120件三种周转率下δ=0.15的总时间均显著更差（约+379/+616/+649s），原“低填充收益/重定位保险”结论被证伪；240件下只观察到失败数减少，但两组仍大量失败，不能称可行性保险。

**② 通俗类比（机制假说，不是已证收益）**：δ相当于给货位打分时留一手“预留金”。这个类比可解释设计动机，但当前生命周期实验不支持把它宣传成性能收益。

**③ 可写进报告的管理语义句**：δ表达“当下摆放效率与未来位置价值”的权衡假说；在当前drain复算中未获得净收益，故保留为待重构机制，不作已验证卖点。

> 注：本轮已完成方法复核，结论是负结果；后续若重做δ策略，需先定义可行域与独立验证集，不能沿用旧“保险”话术。
