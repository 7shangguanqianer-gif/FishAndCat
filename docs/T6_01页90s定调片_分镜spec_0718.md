## 0. 目标与调性
- **一句话**:90 秒把「空仓 0→267 逐件放置,SCORE 为『取』而存」讲成一条有呼吸的定调片,让评委第一眼记住「高频近格 = 取货快」。
- 调性:克制、工业、数据可信(非炫技);ABB 蓝/红克制点缀;字幕少而准,带 (sim) 口径。
- 参数:1280×720、30fps、90s ≈ 2700 帧;DPR 1.5;ANGLE d3d11(同 render_s3_video.mjs 现有约定)。

## 2. 关键帧锚点(01 页 playback API)

- S1 event 0(空仓);S2 event 1(首件 G001,reserveNote 出卡);S3 events 30/60/90/134(金顶热门陆续);S4 保持 event 134 静帧切左栏三联+diffMap;S5 event 267(满仓)+traceStats;S6 event 267 定帧。
