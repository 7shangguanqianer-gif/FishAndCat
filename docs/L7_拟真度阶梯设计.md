## 2. 关键设计决策(实施前提)

1. **档 3 不动头条**:H 锁与回归锁全部不变。档 3 定位=「全拟真验证章」——证明主口径的结论(策略排序、降幅方向、违规 0)在最真实场景下依然成立。这样避免口径变更全流程(拍板→canonical→07/08→全量回归)对头条的冲击,实施成本降一个量级。
2. **分层成本纪律**:档 3 第一版只叠加**负载/运维层**四项(装卸常数、泊松到达、频次噪声、故障事件)= 纯仿真侧,PLC 零改动,不触碰双实现同步铁律。空间结构类(I/O 双口分离、异构仓位)= 档 3.5 扩展,做不做等档 3 结果再议;S 曲线 = 仅敏感性附注(F15 已证物理缩放不变性,"知道下一层在哪且验证过没必要"本身是叙事)。
3. **参数纪律**:档 3 全部新参数(T_load/T_unload、泊松强度 λ、噪声水平、MTBF/MTTR)进 canonical 新增 G 节,初值标「默认待确认」,deep-research 回填后升级为「行业典型值(来源:研究报告)」;一切固定 seed。
4. **展示形态**:同一组核心指标(C3 期望取货 / C8 违规 / C9 吞吐)× 三档阶梯表 + 阶梯图(fig16),报告新章"拟真度阶梯:从数学模型到工业场景"。叙事主张 = **R7 口径不变性的形状化升级**:"结论方向三档不变,绝对数逐档更真——每加一层现实,优势变小但不消失,这正是可信的形状"(预期:档 3 相对降幅小于档 2,提前声明,防评委反问)。
5. **与已定决策的关系**:评分函数四项结构不动(§6 不翻案)——brainstorm 方向 9(订单波次聚簇)与之冲突,**不实施**,进决赛展望一句话;方向 10(不加)被用户"充分采纳"实质否决,但其精神保留于"档 3 不动头条"决策中。
