## 6. 关键教训(0716 下午新增)

1. **elementsFromPoint 会跳过 pointer-events:none 的覆盖物**——查"canvas 被盖"必须几何枚举全元素;重定位 absolute 元素必须显式清空四边(right/top/left/bottom)。
2. **AB ST 编码三查**:UDINT 数组别读进 INT;字符串字面量全 ASCII(中文=乱码警告);新转换函数(如 REAL_TO_STRING)先 grep 全库有无先例。
3. 探针管线(`run_probe.ps1 -Script dump_errors.py`)可 dump 全量编译消息——比 sync 汇总日志多出具体错误行。
4. 大规模 CSS 覆盖层作用域用 `html[data-*]` 开关+显式 grid-column,防母版 grid-area 残留把元素挤进隐式网格。
