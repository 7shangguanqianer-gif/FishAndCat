## 4. 过程中挖出并修掉的 bug

① **CSS 插入破坏 media 块**(自伤自纠):在大屏 media 块内锚点后插顶层规则,须先闭合再重开 media——
  插完立即读回验证(0718 教训:注释吞规则=CSS 静默失效),发现尾部残留空 media 块,清除。

② **01 QA 掉到 45/46(noRuntimeErrors)**:01 的 QA 是**快照制**(把文件清单复制到 hash 快照目录再服务,
  防测试期间源变动),新 css 不在清单 → 404。且其 mime 表**根本没有 `.css` 条目**。
  修:`liveFiles`/`snapshotTargets` 补 `shellCss` + mime 补 `.css`。→ 46/46 回绿。
  **教训:引入新静态依赖时,所有快照制/白名单制的 QA 工装都要同步登记**——404 不会在开发服务器上暴露,
  只在快照服务里炸。
