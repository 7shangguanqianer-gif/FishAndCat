# 02 页功能零基审计 · 2026-07-19

> 用户指令:「重做还不够彻底,深度思考:现有功能哪些真正需要,哪些还缺少」。
> 方法:**零基预算**——每个功能块从"默认删除"出发,必须回答「向评委证明什么」才赢回位置;
> 判断基准 = 评审场景(评委几分钟内要判断:算法真的优吗?证据可信吗?工程能落地吗?)。
> 版面方向已定:方向一(监控作业台,V2 格局精修)。本审计决定"精修"里砍什么、补什么。
