## 3. 四差异标记设计(让评委看出 02≠01,依据 项目关系全景+呈现研究)
1. **取货 lane 高亮**:现有五 lane 里把「取货流」lane 用取货色(红系)描边/高亮,与入库 lane(蓝系)区分;lane 头加「取货侧」标。
2. **双命令「存→取」交织腿高亮**:用现有 trace 事件序,在 cycle 轴/2D 上把「一次行程内 存→取」的交织腿用醒目线型(实线载货/虚线空驶),标出「一趟做两件」。
3. **「省一趟空跑 ~33%」量化角标**:静态角标,引 Bozer-White 1984(单命令两次往返 2×1.27=2.54 vs 双命令一次 1.71,省约 33%);放取货实测块旁或 cycle 轴末。**用业界公认数字,非自造**。
4. **01↔02 能力对比总览(小倍数并置)**:页顶或收尾一个小面板,并排列「01 放货决策 / 02 存取运营」关键指标(存/取、单/双命令、决策层/运营层),颜色标 02 多出的「取货侧」——评委不用脑内做差值。这就是用户要的「同屏可比」。
