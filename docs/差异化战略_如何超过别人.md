# 差异化战略:剩下的五周买什么(对"7 天就完成了,怎么超过别人"的正面回答)

> 2026-07-03,Fable 5。这是战略文档,拍板项见文末;拍板后结论回写 现状与任务.md §6。

## 1. 先回答问题本身

"按目前进度 7 天完成全部"是对的——**完成的是'能交'版**:算法能跑、数字能看、ST 能编译、报告能写满。
全国大多数队伍会在截止前一两周赶出的就是这个版本。7 天版的我们 = 别人的终点。

所以剩余五周不是"把同样的事做慢一点",是**用别人没有的时间预算,买别人买不起的六样东西**。
语音赛已经验证过评委偏好(获奖理由引用的是成熟组合+工程质量+报告质量,不是算法花哨度),
本赛道评分表也是同一结构:模型+报告 30 / PLC 30 / 可视化 20 / 演讲 10 / 创新只有 10。
超越的路径不是更炫的算法,是**每一个评分项都做到评委见过的天花板**。

## 3. 六个提升杠杆(按 ROI 排序,每个独立成活、不动主线口径)

### L1 物理真实度:梯形速度曲线(加减速)+ 装卸时间 【W1-W2,评分项:模型准确性】
- 现状:匀速模型(学生通病)。真实堆垛机加减速 ≈0.3-0.5 m/s²,短程行程被匀速模型显著低估,
  且会改变"近位 vs 低位"的最优解结构——这是实质建模升级,不是装饰。
- 做法:t(d)=分段(三角形/梯形速度曲线),Python 与 FC_CalcTravelTime 同步加 `xUseAccel` 开关,
  口径走 canonical 变更流程,新旧模型对比图一张(又一个"懂行瞬间")。
- 题面依据:"贴近实际工程应用"是原话。

### L2 作业真实度:出库任务流 + 存取混合(双命令周期) 【W2,评分项:模型准确性+创新】
- 关键发现:题面写的是"存取效率(如单位时间**存取**量)"——**取也在题面内,我们目前只做了存**。
  补出库仿真不是加戏,是把题面覆盖完整;大多数队会漏掉这半边。
- 做法:出库任务按频次抽样生成 → 存取混合作业流 → 引入 AS/RS 标准概念"双命令周期"
  (一次行程完成一存一取,省空驶)→ 指标加"单位时间存取量(吞吐)"。
- 这一步做完,"访问频次"才真正闭环:高频货放近处的收益在出库流里直接兑现。

### L3 理论严谨度:解析下界 + Pareto 前沿 【W2-W3,评分项:模型准确性+创新10%】
- NSGA-II 前沿(已拍板做,W-004):AWRA-LS 离前沿多近,一图定音。
- 解析对比:AS/RS 经典文献(Bozer & White 连续货架近似)有期望行程时间解析式,
  把我们的仿真值和理论值并排——"仿真-理论-实现"三级证据链,学术味最足的一页。

### L4 PLC 工程深度:实测扫描负载数据 【W3,评分项:AC500 实现 30%】
- AB 仿真里实测:单周期分片计算耗时(µs)、看门狗裕量倍数、不同 nBatchPerCycle 的负载曲线,
  给出"10ms 任务周期下 CPU 占用 X%"的实测表。
- 学生作品给这种数据的凤毛麟角;这是 ABB 工程师评委最认的语言。

### L5 绿色工程叙事:能耗的工程换算 【W3,评分项:报告+演讲;评委明示偏好】
- 现状:能耗=kg·m 相对值。升级:按假设作业强度(如 500 存取/日)换算 kWh/年、电费、CO2,
  假设全部显式标注(不编数纪律);呼应出题方(ABB 运动控制事业部=卖变频器/电机的)
  "节能增效、脱碳"背景叙事——王玉总监专访点名"绿色工程视角、降低系统能耗"。
- 报告封面级卖点候选:"较基线年省电 X kWh(按声明假设)"。

### L6 演示冲击力:堆垛机动画回放 + 对比模式 【W4,评分项:可视化 20%+演讲 10%】
- 从"网格变色"升级为:堆垛机沿路径逐格移动动画(按切比雪夫时序回放)、
  策略一键切换对比、超重报警现场演示(F3 现象现场复现)。
- 决赛已定 8/15 与初赛全套一起交,这 20% 是六杠杆里单位分值最高的。
