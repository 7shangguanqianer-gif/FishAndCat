老师您好!我们是参加智能仓储赛题的学生队伍,关于提交材料有三个问题想跟您确认一下,每个问题后面附了我们目前的理解,如果理解有偏差麻烦您指正:

**1. 关于初赛决赛材料一起提交的具体方式**
之前了解到初赛想拿高分可以把初赛和决赛内容一起提交。想确认:是在同一个提交包里分目录放齐"初赛三项(建模及仿真、验证报告 PPT 与 WORD、AC500 编程代码)+决赛两项(可视化仿真编程实现代码、演示录屏)"就可以吗?目录组织有没有指定格式?

**2. 关于"可视化仿真的编程实现"的技术形式**
我们自研了一套 Web 版三维可视化仿真(HTML + JavaScript/WebGL 编写,双击浏览器即可打开,无需安装任何软件、不用联网),用来展示仓库布局优化的三维回放、数据看板和演示视频。请问决赛的"可视化仿真的编程实现代码及演示录屏"接受这种自研 Web 形式吗(我们会同时提交全部源码和操作录屏)?还是限定用某类工业软件?另外演示录屏的时长和格式有没有要求?

**3. 关于提交包体积与提交方式**
因为包含演示视频,我们的提交包预计几百 MB 到 1GB。请问总体积有上限吗?最终通过什么方式提交(网盘链接/邮箱/指定平台)?
*我们的预设理解:如有体积限制,我们会优先压缩视频码率保证内容完整。*
