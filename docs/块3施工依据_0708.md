## 1. 五字段输入(T3.1 直接施工配方)

| 字段 | 控件配方(官方一手) |
|---|---|
| 序号 INT/重量/频次/体积 REAL | Rectangle + Input configuration → **OnMouseClick**(非 OnMouseDown!)→ Write a Variable → Input type=`VisuDialogs.Numpad`;**Min/Max=拒绝式**(超范围不写入,非钳位) |
| 名称 STRING | 同上,Input type=`VisuDialogs.Keypad`(字母数字虚拟键盘) |
| 名称下拉+可手输 | **无现成控件**,拼装:`Combo Box, Array` 绑 `ARRAY OF STRING`(选中写 **INT 下标**,非字符串)→ ST 侧 `IF iSel>=0 THEN stName:=astrNames[iSel]` 同步 → 同一 stName 叠加 Keypad 文本框 |
| 校验反馈 | 首选 Min/Max 硬拒绝;要"看得见错"用 Input type=`Text input with limits`(超限数值文本变红);"整块变红"需自拼(Fill color 绑判断表达式) |

依据(逐字核实):content.helpme-codesys.com `_visu_dlg_input_configuration.html`、
`_visu_configure_text_input_with_virtual_keyboard.html`(标题级)。

## 2. Alarm Banner 报警条(T3.2 配方)

配置链(★=逐字核实):建 Alarm Configuration 对象 → Alarm Class(优先级/确认方式/颜色)→
Alarm Group 内建 Alarm★:Observation Type=**Digital**★,Expression=`GVL_Visu.VisuAlarm`★,
条件 `=TRUE`★ → 画面拖 **Alarm Banner** 元素★ → 属性勾选该 Group/Class★,Filter=Newest,
列=State/Message/Timestamp → 确认按钮:BOOL 变量绑 Acknowledge 属性★。

**重大澄清(解除 0708 工业枪的警示)**:官方"the use of trends or alarms …not provided in an
**integrated visualization**"里的 integrated visualization=**IDE 在线预览视图**(交叉核实
`_visu_f_overview.html`),**≠WebVisu**。即:AB 里在线预览时 Alarm Banner/Trend 不渲染是正常
的,部署到 WebVisu/TargetVisu 应可用。⚠️ 残留风险:早期 CODESYS SP 版本 WebVisu 对 Alarm/Trend
支持不全的历史限制——**开工验证项 V3**(见 §5)。

## 3. T3.4 实时联动通道(外部程序周期读 400+ 变量)

| 通道 | 判定 | 要点 |
|---|---|---|
| **Modbus TCP(推荐)** | 长期方案 | PLC 侧 ETH 下加从站+映射寄存器,固件处理协议栈;Python 用 pymodbus,与 ScriptEngine 管线完全解耦;⚠️ 单请求 125 寄存器上限(协议惯例)→ 400 INT 拆 4 次,非原子快照(堆垛机坐标可能撕裂,可 PLC 侧打包窗口缓解);AC500 从站栈实现细节待下轮查 library.e.abb.com |
| OPC UA Server(备选) | V3 原生 | ETH 下加 OPC UA Server 对象+Symbol Configuration 勾"Support OPC UA Features",端口 **4840**;数组=整节点可订阅;性能指标未查到(App Note 3ADR010661 PDF 待读) |
| ScriptEngine 常驻会话 | **仅过渡/演示** | `read_values()` 批量 API 存在,但数组整读行为/轮询频率/长会话稳定性**官方零背书**;与 build/test 管线抢 Gateway 未验证 |

ScriptEngine 三步实测方案(若走过渡路线):①3-4 元素子数组试 `read_values(["GVL.VisuSlotState"])`
整传 vs 展开传,看返回结构 ②time.time() 包 402 变量调用测真实往返 ③2-4 小时长跑看内存/连接漂移。

## 4. WebVisu 发布(T3.4 配方)

步骤(逐字核实 `_visu_execute_webvisu.html`):Visualization Manager → Add Object →
**WebVisu Object** → 配起始画面+htm 文件名 → Show Used Visualizations 激活 → 编译下载启动
(**Web 服务器随应用自动启停**)→ 浏览器 `http://<PLC_IP>:8080/<htm名>`。

⚠️ **开工验证项 V4(最重要的未闭环)**:AC500 的 WebVisu 是否需要许可——ABB 许可分
Basic(免费)/Standard/Premium 三档,可视化运行时属哪档**未查实**;在 AB2.9 许可对象或
AC500 datasheet 核实。若收费档→WebVisu 降级为"演示 PC 上 CODESYS HMI/在线预览"。
