# 01/03 一屏驾驶舱返工规格(0722 九问拍板,总纪律沿用 docs/施工规格_回灌0721.md §0)

> 拍板依据:docs/3版拍板包_0721.md「0722 返工拍板(九问)」节。排查结论:面板内部滚动是滚动主因(document 级不滚断言是漏洞);3D 镜头未按容器自适应;「继续」按钮系错删须以自动播放+暂停钮替代。

## §A 一屏纪律(两页共同,机器断言)

1. 设计基准 1920×1080;1280×800 下允许块收纳但**仍零滚动**。
2. **零滚动定义(新 QA 断言,双分辨率各测)**:document 与所有 overflow-y 容器 scrollHeight ≤ clientHeight+2。白名单:弹层内部(90vh 内可滚)、显式横向带(overflow-x)。
3. 弹层规范:遮罩+居中大层(max 90vw/90vh),Esc/点遮罩关闭,内部可滚。
4. 截图底座(所有 QA/验收采用):`screenshot({animations:'disabled', caret:'hide'})`;语义就绪断言替代 networkidle/固定 sleep(等目标元素可见/计数/文本);交互序列=操作→状态断言(aria-expanded/hidden/尺寸)→截图;`document.fonts.ready` 后再拍;1920×1080 与 1280×800 双分辨率。

## §B 01 页(3D 主角+赛马精华条)

1. **版面**:3D 全高主角(右侧 ≥65% 或全宽底层);左/下「赛马精华条」单条常驻:三联缩略图(小,可点)+ 当前容量节点三算法关键指标微条 + SCORE 相对 SEQ 领先徽章。口径声明压缩为精华条头部一行小字(hover 展开全文)。进度轴保留(顶部细条)。**赛段比分条全量表/差异热力图/评分解剖/终态总分牌全部撤出常驻区**,收入弹大层。
2. **弹大层**(精华条点击唤起,Tab 或分区):三联大图+差异热力图 / 赛段比分全表+终态总分牌 / 评分解剖(瀑布+Top-3)。审计抽屉维持右上「数据链已验证」入口。30-seed 分布带入总分牌区。
3. **播控**:进页自动播放(就绪后自动 play,与 02 行为对齐);顶栏暂停/继续切换钮(单钮双态,aria-pressed);容量节点点击跳转保留。倍率滑条保留。
4. **3D 全景自适应**:镜头按容器宽高比+货架包围盒自动拉距,保证货架全高+入库口入镜,任何容器尺寸不截顶(优先找 runtime 现成 resizeScene/restoreDefaultCamera/fitCamera 改造,勿另起炉灶)。折叠/窗口变化后 resize 必须重算镜头。
5. **1280×800 收纳**:精华条降单行、顶栏元素合并,3D 不小于 55% 宽,零滚动。
6. **QA 适配**:53 项中版面相关断言改语义;新增:零滚动双分辨率×(01+弹层白名单)、自动播放已开始(simTime 增长)、暂停钮双态、镜头不截顶代理断言(canvas 高≥容器高 95%+人工图审)、弹大层开合、精华条存在。**契约 30+02QA 8 不回归。**

## §C 03 页(顶部 4 Tab 分区)

1. **Tab**:证据画面 / 证据链 / I/O 矩阵 / 边界与负结果。顶栏下 Tab 条(与 01 精华条同高度节奏);默认 Tab=证据画面。口径带保留(压缩单行)。
2. **每 Tab 一屏零竖滚**(§A 断言):证据链 9 卡改 3×3 网格(卡压缩:主张+缩略图+路径一行+徽章,点卡弹大层看全档);I/O 矩阵 Tab=矩阵表+判定方法折叠+六站点表(点站弹点表大层);边界负结果=七条双列网格(措辞一字不改红线);证据画面=三视图 Tab+元数据卡(现有,天然一屏)。40 帧带+视频归入证据画面 Tab 底部横向带(横向白名单)。
3. **QA 适配**:15 项保留语义;新增:Tab 四区切换各拍、每 Tab 双分辨率零滚动、弹大层开合。负结果逐字断言沿用。
