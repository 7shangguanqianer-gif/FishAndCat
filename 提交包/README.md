## 摘要

针对 20×20 自动化立体仓库的存取货效率问题,本作品提出并实现了自适应加权与局部
搜索相结合的布局优化方法(AWRA-LS),配合场景检测器在多目标档位间自动路由。
系统完成算法(Python 统计验证)→ 控制器(ABB AC500 ST 全量实现)→ 人机界面
(三页制 HMI)→ 物理级验证(Factory I/O 全链取放)的完整工程链路。

## 主要结果

| 指标 | 数值 | 验证方式与口径 |
|---|---|---|
| 期望取货时间降幅(vs 顺序基线) | −61.6% | 30 随机种子统计,8.17±0.67 s(sim) |
| 控制器在线自测 | 77 / 0 | AB PC 仿真在线读回,全真实断言 |
| 物理全链取放验证 | 9 / 9 | Factory I/O 试验台,3 库位×3 次,零故障 |
| 场景检测器 oracle 达成率 | 98.5% / 99.19% | 390 个可比实例(sim,词典序档);匀速 / 加减速双口径 |

注:凡标注 (sim) 均为计算机仿真统计,非实体设备实测;物理验证采用商业虚拟调试
软件的代表性场景(54 存储位)完成。完整口径与统计方法见验证报告。
