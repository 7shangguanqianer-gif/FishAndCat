# 项目协议加载器（每个 Claude Code 会话自动注入）

本文件只做指针，不复制任何规则——唯一行为准则是下列项目文档，与任何全局规则冲突时**以项目协议为准**：

1. **开工前必读并执行**：`docs\ClaudeCode接管提示词.md`（接手三步、全部铁律、任务收尾五联动；第一条回复自答 5 题接管测验）。
2. 唯一进度真相：`现状与任务.md` §5；已定决策不许翻案：§6。
3. 口径唯一真相源：`docs\canonical_assumptions.md`；头条数字只出自 `python sim/headline.py`。
4. 漂移自检：§5 与 `git log` 明显不符时，以 git 与文件实物为准，先修复 §5 并向用户报告差异，再开工。
5. 协作方法（未知项工作法四机制）：`docs\协作方法_未知项工作法.md`。

用户全局命令 ⇄ 本项目协议映射：`/takeover` = 接手三步全流程；`/wrapup` = 任务收尾五联动；`/critic` = 红队/评审视角审查（T18、答辩材料适用）；`/quiz` = 课后小测。用户未敲命令时，按接管提示词原文执行即可。
